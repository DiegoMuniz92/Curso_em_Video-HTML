conteiner -> conteiner fixo.

conteiner-fluid -> ocupa 100% width.

pt/pb- -> adiciona um padding top/botton

border -> adiciona uma borda padrao.

bg- -> adiciona uma cor de fundo.

text- -> altera cor do texto.

p- -> adiciona um padding em todos os lados

GRID
    - row -> para informar a linha do grid (sempre necessario para iniciar um grid)
    - col -> para informar as colunas
    - col-1 -> para altera o tamanho individualmente, quanto maior o valor maior o tamanho do elemento.

FONTES
    - class="h1" -> propriedade de tipografia.
    - display-1 -> propriedade de tipografia com peso diferente.
    - cmall -> para seu funcionamento e necessário que o elemento esteja envelopado pela tag span.

ALINHAMENTO
    - d- -> altera o tipo de display
    - mx/my -> alinha o elemento na horizontal/vertical

CORES
    - text-white -> exemplo de cor predefinida
    - bg-white -> exemplo de cor de fundo 
    - opacidade -> unico valor disponivel (50), exemplo:  text-white-50

TABELAS
    - table -> para criar a estrutura de tabela, tbm pode ser usado nas linhas e colunas da tabela.
    - table-dark -> para inserir cor de fundo
    - table-striped -> para criar um efeito de claro e escuro entre as linhas.
    - table-hover -> para criar o efeito ao passar o mouse.
    - table-bordered -> para criar as bordas
    - table-borderless -> para criar uma tabela sem bordas

IMAGENS
    - para usar imagens basta inserir o w- com alguns valores padrões (25/50/75/100), em imagens png é possível alterar a cor de fundo.
    - rounded -> propriedade utilizada para as bordas de uma imagem.
    - img-thumbnail -> criar uma borda ao redor da imagem 
    - img-fluid -> transforma a imagem em responsiva ("max-width: 100%, height: auto")
